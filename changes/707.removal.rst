The following previously-deprecated functionality has been removed:

  - ``dqflags.interpret_bit_flags`` has been removed. Use
    ``astropy.nddata.bitmask.interpret_bit_flags`` instead.
  - ``DataModel.from_asdf`` has been removed. Use ``DataModel.__init__``
    directly instead.
  - ``DataModel.from_fits`` has been removed. Use ``DataModel.__init__``
    directly instead.
  - ``DataModel.get_fits_wcs`` has been removed. To get the SIP
    approximation, call
    ``to_fits_sip()`` on ``model.meta.wcs`` instead.
  - ``DataModel.set_fits_wcs`` has been removed. Modify ``model.meta.wcs``
    directly instead.
  - The ``memmap`` keyword argument to ``DataModel.__init__`` and
    ``datamodels.open`` has been removed.
    Memory mapping is no longer supported.
  - Passing ``**kwargs`` through ``asdf_in_fits.open`` and
    ``fits_support.from_fits_asdf`` into
    ``asdf.open`` is no longer supported.
  - Passing ``None``, a shape tuple, a ``numpy.ndarray``, or an
    ``astropy.io.fits.HDUList`` to ``datamodels.open`` now raises
    ``TypeError``. Use the ``DataModel`` constructors directly instead.
  - Passing unrecognized keyword arguments to ``DataModel.__init__`` when
    ``init`` is file-like now raises ``TypeError`` instead of
    a deprecation warning.
